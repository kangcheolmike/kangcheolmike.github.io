주요 기능 — 뮤직봇 핵심

음악 재생에 필요한 모든 것을 빠르고 간편하게 제공합니다.

초고음질 스트리밍

FFmpeg 최적화·고비트레이트·자동 음량 정규화로 또렷한 사운드.

강력한 이퀄라이저 & 필터

베이스 부스트, 트레블 업, 나이트코어, 8D 등 다양한 프리셋.

재생목록·대기열

개인/서버 재생목록 저장, 셔플·반복·건너뛰기 모두 지원.

슬래시 명령어 100%

/재생부터 /이퀄라이저까지 직관적인 사용성.

슬래시 명령어

핵심 명령어 — 빠르게 붙여넣어 쓰기 좋은 형식으로 준비했습니다.

▶ 재생 관련

/재생 [쿼리]
유튜브 URL 또는 검색어로 노래를 재생합니다. 이미 음악이 재생 중이라면 대기열에 추가됩니다.
/검색 [쿼리]
검색어로 유튜브에서 최대 5개의 결과를 가져옵니다. 원하는 노래를 선택해서 재생 가능합니다.
/재생목록 [url]
유튜브 재생목록 URL을 입력하면 여러 곡을 한 번에 대기열에 추가합니다.

⏯ 재생 제어

/정지
음악을 멈추고 대기열을 초기화합니다.
/스킵
현재 곡을 건너뛰고 다음 곡을 재생합니다.
/일시정지
음악을 일시정지합니다.
/재개
일시정지된 곡을 다시 재생합니다.
/이동 [초]
현재 곡에서 입력한 시간(초)으로 건너뜁니다.

📜 대기열 관련

/대기열
현재 재생 대기열을 보여줍니다.
/삭제 [번호]
대기열에서 특정 번호의 곡을 제거합니다.
/셔플
대기열을 무작위로 섞습니다.

🎶 음악 설정

/지금재생
현재 재생 중인 곡의 정보를 보여줍니다.
/루프 [모드]
반복 모드를 설정합니다.
  • 끄기 : 반복 해제
  • 한 곡 반복 : 현재 곡만 무한 반복
  • 대기열 반복 : 대기열 전체를 반복
/볼륨 [레벨]
재생 볼륨을 1 ~ 100 사이로 조절합니다.
/이퀄라이저 [모드]
음질을 바꿉니다. 예: 기본 / 베이스 부스트 / 팝

시작하기 — 뮤직봇 빠른 가이드

  1. 위의 봇 초대하기 버튼으로 서버에 초대합니다. 권한은 연결/말하기/메시지 보기가 필요합니다.
  2. 보이스 채널에 접속한 뒤 텍스트 채널에서 /연결을 실행합니다.
  3. 원하는 곡을 /재생 검색어 또는 URL로 재생합니다.
  4. 사운드를 꾸미고 싶다면 /이퀄라이저와 /필터를 활용하세요.
문제 발생 시 지원 서버에서 도움을 받으세요.

자주 묻는 질문

음악이 끊겨요 / 지연이 커요.

서버 리전 자동선택, 다른 음성봇/스트림 종료, 비트레이트 96kbps 이상 권장. 지속 시 지원 서버로 로그를 보내 주세요.

저작권은 어떻게 되나요?

봇은 링크 재생/검색을 중개할 뿐, 음원 파일을 저장하지 않습니다. 사용 시 각 플랫폼의 정책을 준수해 주세요.
